# APA YANG BARU PADA MIzano V.3 ?
;------------------------------------------------------------------------------;
Jika Versi sebelumnya adalah MIzano antivirus BETA 1.0.1 & 1.0.2 ,YANG SEKARANG TELAH SAYA 
kembangkan lagi menjadi Version 3 atau V.3 dan saya juga merilis ulang coding 
yang lama dari awal hingga jadilah MIzano AntiVirus ..Oke Kelebihan Dari MIzano 1.0.3 Adalah :

1. Power Removal : Dapat memonitor proses komputer .di sini user jga dapat menghentikan proses secara lansung dengan menekan tombol " Terminate "

2. Shield [RTP]  : Dengan adanya fasilitas ini user dapat dimudahkan dengan pendeteksian virus secara otomatis / Real Time Scanning .Shield yg terbaru ini
                         saya rilis ulang dari awal , insya allah Program ini dapat mendeteksi virus secara cepat 

3. Firewall : Program ini mampu memonitor koneksi yg berhubungan dengan komputer anda .dan program ini juga mampu menscan proses2 yg sedang di monitor olh firewall Tersebut
              Firewall juga dapat mendeteksi port-port yg terbuka oleh program2 yang mencurigakan 

4. Registry Fixer : Fasilitas ini adalah Bengkel Registry , Dimana user dapat membetulkan registry2 yg error atau registry2 yg dimanipulasi oleh virus
